# use-sidebar-sections.ts — Brief + Junior Q&A

**Path:** `apps/web/lib/use-sidebar-sections.ts`  
**Role:** Filter sidebar nav by role + billing plan flags.

---

## Junior Q&A

--------------------------------------------------------------------------------------------------------------------------------------------
**[0] What does `useSidebarSections` return?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Filtered array of `SidebarSection` for `AppLayout` — `use-sidebar-sections.ts:85-104`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[1] What inputs affect the sidebar?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** `useLayoutUser()` role, `GET_TENANT_BILLING` feature flags, guest vs logged-in — `86-97`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[2] What can a guest see?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Only items with ids: dashboard, listings-directory, my-listings, profile, settings — `33-35`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[3] Who can see Customers / Orders?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Admin+ roles — `filterItem` cases `customers`, `orders` — `38-42`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[4] Why `useMemo`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Avoid re-filtering ~50 nav items on unrelated re-renders — deps `99-104`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[5] Is this secure by itself?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** **No** — UI only. API must enforce same rules.
