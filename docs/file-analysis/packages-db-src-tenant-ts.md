# tenant.ts — Brief + Junior Q&A

**Path:** `packages/db/src/tenant.ts`  
**Role:** Mongoose model for organizations (multi-tenant root).

---

## Junior Q&A

--------------------------------------------------------------------------------------------------------------------------------------------
**[0] What is a tenant in LuxGen?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** One customer org — has `subdomain` (`demo`), `status`, embedded `settings` (branding, security).

--------------------------------------------------------------------------------------------------------------------------------------------
**[1] Why is `subdomain` unique?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** URL routing: `demo.localhost` → find tenant by subdomain — schema `107-114`, index `292`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[2] What is `mongoose.models.Tenant || model(...)`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Prevents `OverwriteModelError` on hot reload — `tenant.ts:295-296`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[3] What indexes exist?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** `subdomain`, `domain`, `status` — `tenant.ts:292-294`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[4] JWT tenant claim vs subdomain?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** JWT stores Mongo **id**; subdomain is for URLs only. Never compare them directly.

--------------------------------------------------------------------------------------------------------------------------------------------
**[5] What query resolves tenant on API boot?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** `Tenant.findOne({ subdomain, status: 'active' })` — see `context/buildContext.ts:16`.
