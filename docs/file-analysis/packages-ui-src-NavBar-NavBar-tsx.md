# NavBar.tsx — Brief + Junior Q&A

**Path:** `packages/ui/src/NavBar/NavBar.tsx`  
**Role:** Top bar — logo, search, theme, user menu or guest Login/Sign Up.

---

## Junior Q&A

--------------------------------------------------------------------------------------------------------------------------------------------
**[0] How does NavBar know if user is logged in?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Optional `user` prop — if missing/falsy, show guest buttons — `NavBar.tsx:21-22`, branch `381-396`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[1] What must pages pass for guest UI?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** `user={undefined}` or omit `user` — never pass a fake default user.

--------------------------------------------------------------------------------------------------------------------------------------------
**[2] What local state does NavBar use?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Mobile menu, tenant menu, mobile search — `NavBar.tsx:72-74`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[3] When does super-admin tenant switcher show?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** `user.role === SUPER_ADMIN` and `tenantSwitch` prop set — `NavBar.tsx:48`, `78`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[4] Where are Login / Sign Up links?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Guest branch — `NavBar.tsx:383-396` (`href="/login"`, `href="/register"`).
