--------------------------------------------------------------------------------------------------------------------------------------------
**[0] What is `let` vs `const`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** `const` cannot be reassigned; `let` can. Prefer `const` unless you reassign.

**In LuxGen:** `AUTH_STORAGE_KEYS` is `const` — `apps/web/lib/session.ts:27-34`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[1] What is a closure?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** A function that remembers variables from its outer scope.

**In LuxGen:** `createHandleUserAction(router)` keeps `router` for logout redirect — `apps/web/lib/user-actions.ts` (search `createHandleUserAction`).

--------------------------------------------------------------------------------------------------------------------------------------------
**[2] What is `===` vs `==`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** `===` compares value and type. Always use `===` in this codebase.

--------------------------------------------------------------------------------------------------------------------------------------------
**[3] What is destructuring?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Pull fields from objects/arrays into variables.

**In LuxGen:** `const { data, loading } = useQuery(...)` — used on dashboard `apps/web/pages/dashboard.tsx:32-40`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[4] What is spread (`...`)?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Copy object/array (shallow). `{ ...headers, authorization: token }` — `apps/web/graphql/client.ts:71-76`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[5] What is optional chaining `?.`**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Safe access — `undefined` if middle is null/undefined.

**In LuxGen:** `data?.orderRows` — `apps/web/pages/orders/index.tsx:77`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[6] What is nullish coalescing `??`**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Default only for `null`/`undefined` (not `0` or `""`).

**In LuxGen:** `layoutUser ?? undefined` — `apps/web/pages/dashboard.tsx` / `_app.tsx:42`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[7] What is a Promise?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Object for async result later — `.then()` / `await`.

**In LuxGen:** `stopHealthServer()` returns `Promise<void>` — `apps/agent-worker/src/health.ts:20-25`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[8] What is `async/await`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Syntax to wait for Promises without callback hell.

**In LuxGen:** API services use `async` resolvers — `apps/api/src/schema/` (search `async (`).

--------------------------------------------------------------------------------------------------------------------------------------------
**[9] What is the event loop?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** JS runs one call stack; microtasks (Promises) run before next macrotask (setTimeout).

**In LuxGen:** After GraphQL login, `persistSession` then `luxgen-auth-change` listeners run in same tick — `session.ts:22-24`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[10] What is `localStorage`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Browser key-value store (persists). Used for JWT + user JSON.

**Read:** `session.ts:27-34`, `persistSession` at `118`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[11] What is JSON.stringify / JSON.parse?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Object ↔ string for storage/API.

**In LuxGen:** `currentUser` stored as JSON string — `session.ts` `getStoredUser` ~`150`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[12] What is a module (import/export)?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Each file is a module; `export` shares code; `import` pulls it in.

**In LuxGen:** Monorepo packages `@luxgen/ui`, `@luxgen/db` imported across apps.

--------------------------------------------------------------------------------------------------------------------------------------------
**[13] What is `Array.map`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Transform each item → new array (used to render React lists).

**In LuxGen:** Order rows mapped — `apps/web/pages/orders/index.tsx:76-78`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[14] What is `Array.filter`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Keep items that pass a test.

**In LuxGen:** Sidebar filter — `apps/web/lib/use-sidebar-sections.ts:60-69`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[15] What is `typeof window === 'undefined'`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Detect server (SSR) vs browser. Skip `localStorage` on server.

**In LuxGen:** `session.ts:107`, `client.ts:56`, `AuthGuard` mount check.

--------------------------------------------------------------------------------------------------------------------------------------------
**[16] What is an arrow function?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Shorter function syntax; `() => {}`. Often used in hooks/callbacks.

**In LuxGen:** `authLink` — `apps/web/graphql/client.ts:55`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[17] What is a template literal?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Backtick strings with `${variable}`.

**In LuxGen:** `` `Bearer ${token}` `` — `client.ts:74`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[18] What is `try/catch`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Handle errors without crashing.

**In LuxGen:** JWT decode — `session.ts:74-82` `decodeJwtPayload`.

--------------------------------------------------------------------------------------------------------------------------------------------
**[19] What is `Set`?**
--------------------------------------------------------------------------------------------------------------------------------------------

**ANS.** Collection of unique values. Used for selected row IDs in bulk UI.

**Search repo:** `new Set(` in automations pages.
