# Button

LuxGen UI component. See `Button.tsx` for props and usage.

## Structure

| File              | Purpose                         |
| ----------------- | ------------------------------- |
| `Button.tsx`      | Main component                  |
| `fetcher.ts`      | Data fetching / SSR helpers     |
| `fixture.ts`      | Test fixtures and mock data     |
| `Button.spec.ts`  | Unit tests                      |
| `styles.ts`       | Styled components and CSS-in-JS |
| `translations.ts` | i18n strings                    |
| `index.ts`        | Public exports                  |

## Usage

```tsx
import { Button } from '@luxgen/ui';

<Button />;
```
