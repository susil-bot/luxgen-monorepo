# CountryLanguageDropdown

LuxGen UI component. See `CountryLanguageDropdown.tsx` for props and usage.

## Structure

| File                              | Purpose                         |
| --------------------------------- | ------------------------------- |
| `CountryLanguageDropdown.tsx`     | Main component                  |
| `fetcher.ts`                      | Data fetching / SSR helpers     |
| `fixture.ts`                      | Test fixtures and mock data     |
| `CountryLanguageDropdown.spec.ts` | Unit tests                      |
| `styles.ts`                       | Styled components and CSS-in-JS |
| `translations.ts`                 | i18n strings                    |
| `index.ts`                        | Public exports                  |

## Usage

```tsx
import { CountryLanguageDropdown } from '@luxgen/ui';

<CountryLanguageDropdown />;
```
