# CourseMenu

LuxGen UI component. See `CourseMenu.tsx` for props and usage.

## Structure

| File                 | Purpose                         |
| -------------------- | ------------------------------- |
| `CourseMenu.tsx`     | Main component                  |
| `fetcher.ts`         | Data fetching / SSR helpers     |
| `fixture.ts`         | Test fixtures and mock data     |
| `CourseMenu.spec.ts` | Unit tests                      |
| `styles.ts`          | Styled components and CSS-in-JS |
| `translations.ts`    | i18n strings                    |
| `index.ts`           | Public exports                  |

## Usage

```tsx
import { CourseMenu } from '@luxgen/ui';

<CourseMenu />;
```
