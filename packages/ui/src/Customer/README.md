# Customer

LuxGen UI component. See `Customer.tsx` for props and usage.

## Structure

| File               | Purpose                         |
| ------------------ | ------------------------------- |
| `Customer.tsx`     | Main component                  |
| `fetcher.ts`       | Data fetching / SSR helpers     |
| `fixture.ts`       | Test fixtures and mock data     |
| `Customer.spec.ts` | Unit tests                      |
| `styles.ts`        | Styled components and CSS-in-JS |
| `translations.ts`  | i18n strings                    |
| `index.ts`         | Public exports                  |

## Usage

```tsx
import { Customer } from '@luxgen/ui';

<Customer />;
```
