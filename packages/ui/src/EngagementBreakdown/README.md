# EngagementBreakdown

LuxGen UI component. See `EngagementBreakdown.tsx` for props and usage.

## Structure

| File                          | Purpose                         |
| ----------------------------- | ------------------------------- |
| `EngagementBreakdown.tsx`     | Main component                  |
| `fetcher.ts`                  | Data fetching / SSR helpers     |
| `fixture.ts`                  | Test fixtures and mock data     |
| `EngagementBreakdown.spec.ts` | Unit tests                      |
| `styles.ts`                   | Styled components and CSS-in-JS |
| `translations.ts`             | i18n strings                    |
| `index.ts`                    | Public exports                  |

## Usage

```tsx
import { EngagementBreakdown } from '@luxgen/ui';

<EngagementBreakdown />;
```
