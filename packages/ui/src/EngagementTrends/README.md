# EngagementTrends

LuxGen UI component. See `EngagementTrends.tsx` for props and usage.

## Structure

| File                       | Purpose                         |
| -------------------------- | ------------------------------- |
| `EngagementTrends.tsx`     | Main component                  |
| `fetcher.ts`               | Data fetching / SSR helpers     |
| `fixture.ts`               | Test fixtures and mock data     |
| `EngagementTrends.spec.ts` | Unit tests                      |
| `styles.ts`                | Styled components and CSS-in-JS |
| `translations.ts`          | i18n strings                    |
| `index.ts`                 | Public exports                  |

## Usage

```tsx
import { EngagementTrends } from '@luxgen/ui';

<EngagementTrends />;
```
