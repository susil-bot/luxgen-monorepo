# GroupCard

LuxGen UI component. See `GroupCard.tsx` for props and usage.

## Structure

| File                | Purpose                         |
| ------------------- | ------------------------------- |
| `GroupCard.tsx`     | Main component                  |
| `fetcher.ts`        | Data fetching / SSR helpers     |
| `fixture.ts`        | Test fixtures and mock data     |
| `GroupCard.spec.ts` | Unit tests                      |
| `styles.ts`         | Styled components and CSS-in-JS |
| `translations.ts`   | i18n strings                    |
| `index.ts`          | Public exports                  |

## Usage

```tsx
import { GroupCard } from '@luxgen/ui';

<GroupCard />;
```
