# RegisterForm

LuxGen UI component. See `RegisterForm.tsx` for props and usage.

## Structure

| File                   | Purpose                         |
| ---------------------- | ------------------------------- |
| `RegisterForm.tsx`     | Main component                  |
| `fetcher.ts`           | Data fetching / SSR helpers     |
| `fixture.ts`           | Test fixtures and mock data     |
| `RegisterForm.spec.ts` | Unit tests                      |
| `styles.ts`            | Styled components and CSS-in-JS |
| `translations.ts`      | i18n strings                    |
| `index.ts`             | Public exports                  |

## Usage

```tsx
import { RegisterForm } from '@luxgen/ui';

<RegisterForm />;
```
