# Timeline

LuxGen UI component. See `Timeline.tsx` for props and usage.

## Structure

| File               | Purpose                         |
| ------------------ | ------------------------------- |
| `Timeline.tsx`     | Main component                  |
| `fetcher.ts`       | Data fetching / SSR helpers     |
| `fixture.ts`       | Test fixtures and mock data     |
| `Timeline.spec.ts` | Unit tests                      |
| `styles.ts`        | Styled components and CSS-in-JS |
| `translations.ts`  | i18n strings                    |
| `index.ts`         | Public exports                  |

## Usage

```tsx
import { Timeline } from '@luxgen/ui';

<Timeline />;
```
